# JobTracker

JobTracker is a personal web application for tracking job applications. The M1
milestone provides the FastAPI and PostgreSQL backend foundation.

## Repository

- `backend/`: FastAPI application, Alembic migrations, and backend tests.
- `database/`: Database ownership notes.
- `docs/`: Product, architecture, ADR, and milestone documentation.
- `frontend/`: Reserved for M2; it is intentionally not created by M1.

See `backend/README.md` for backend setup and API usage.
