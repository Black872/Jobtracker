# Codex Instructions For JobTracker

You are responsible for application development only.

Focus on:
- Frontend
- Backend
- Database
- REST API
- Testing
- Documentation
- Business logic

Do NOT implement:
- Docker
- GitHub Actions
- Kubernetes
- Helm
- Terraform
- Ansible
- ArgoCD
- GitOps
- Monitoring
- Infrastructure provisioning
- VPS configuration

Always follow:
- ADR
- Technical Specification
- Architecture
- Roadmap

Before implementing features:
1. Explain the approach.
2. Explain alternatives.
3. Recommend one solution.
4. Wait for approval if architecture changes are significant.

Prefer helping the user learn.
Keep solutions simple and maintainable.
Do not automatically generate infrastructure code.
